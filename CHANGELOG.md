# Changelog

All notable project changes should be recorded here.

The format is based on Keep a Changelog, with release tags anchored in git.

## [Unreleased]

### Added

- Release workflow documentation for `master`, `develop`, and short-lived work branches.
- `VERSION` as a single source of truth for the current release number.
- `scripts/git-start-branch.sh` to start new work from the right base branch.

### Changed

- Polished the presentation shell with a stronger presentation header, a richer opening slide, and a more complete closing slide.

### Fixed

- Ignore Python bytecode and `__pycache__` artifacts so generated files stop polluting history.

## [1.0.0] - 2026-04-04

### Added

- Public open-source release of the interactive paper, presentation mode, and Win95-style hardware experience.
- Fullscreen presentation scenes covering the title, methods, scope, connections, implications, advocacy, preparation, and finale.
- Interactive demos, explorer content, Bonzi assistant flows, and local AI proxy support for self-hosted use.

### Changed

- Consolidated project docs, screenshots, and release-facing repository metadata.
- Switched the repository license to AGPLv3.
