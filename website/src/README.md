# Website Runtime Structure

This folder contains the runtime JavaScript split by responsibility.

## Layout

- `shared/`
  - `win95-shared.js`: Cross-feature helpers (safe storage wrappers, file metadata labels/icons).
- `runtime/`
  - `main.js`: Core 3D scene, boot flow, desktop/window manager, and shared globals.
  - `modules/`
    - `icon-drag.js`: Desktop icon grid layout, drag/drop snapping, and persistence.
    - `context-menu.js`: Wallpaper context menu wiring and related desktop actions.
    - `presentation-animations.js`: Presentation panel slide/scroll animation orchestration.
- `features/`
  - `games-shared.js` + `games-snake.js` + `games-runner.js` + `games-race.js` + `games.js`: Split mini-game modules and API aggregator.
  - `explorer-fs.js` + `explorer.js`: Split file-system data model and explorer UI/runtime logic.
  - `steam-game-*.js` + `steam-catalog.js` + `steam.js`: One-file-per-game definitions, catalog aggregation, and Steam95 launcher/runtime UI.
  - `bonzi-core.js` + `bonzi.js`: Bonzi shared config/helpers and controller runtime.
  - `extras-windowing.js`: Window-manager enhancements (resize handles, snap, minimize/restore patches).
  - `extras-app-minesweeper.js` + `extras-app-paint.js` + `extras-app-internet-explorer.js` + `extras-app-msn.js` + `extras-app-shutdown.js` + `extras-app-system-properties.js` + `extras-app-winamp.js`: One-file-per-app modules.
  - `extras-system-screensaver.js`: Standalone screensaver/idle system module.
  - `extras-apps-a.js` + `extras-apps-b.js`: Compatibility/runtime aggregators.
  - `extras-apps.js`: Legacy wrapper retained for compatibility reference.
  - `extras.js`: Legacy monolith retained for compatibility reference (not loaded by default).
- `presentation/`
  - `config.js`: Presentation constants, profiles, and narrator script tables.
  - `particle-shapes.js`: Morph target buffer generation for transition particle systems.
  - `transition-engine.js`: Slide navigation and transition orchestration.
  - `effects/`: Reusable transition flavor and content-motion presets.
  - `scenes/`: Per-slide scene controllers (`slide-01` through `slide-09`) for isolated interactions.
  - `presentation-mode.js`: Fullscreen presentation orchestrator/controller.

## Notes

  - `src/presentation/index.js` is the bootstrap loaded by `index.html`, which instantiates `PresentationMode`.
- Source file indexing is generated by `website/scripts/sync-explorer-index.js`.
