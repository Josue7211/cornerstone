WEB AUDIO SECTION (Web Audio API — no external files, all synthesized):

Function getAudioCtx(): lazy-initializes a single AudioContext stored in a closure variable.
Returns the AudioContext.

Function playClickSound(): Creates oscillator + gain node. Frequency sweeps from 1200 to 600 Hz
over 0.06 seconds. Gain envelope: 0.15 at start, decays to 0.001 at 0.08s. Wraps in try/catch.

Function playStartupChime(): Plays 4 tones as a chord sequence with slight timing offset.
Frequencies: 523.25, 659.25, 783.99, 1046.50 Hz (C5, E5, G5, C6 — Win95-like ascending chime).
Each tone: sine wave, gain ramps from 0 to 0.18 in 0.02s, decays to 0.001 at 0.35s.
Tones staggered by 0.12 seconds. Wraps in try/catch.

Function playWindowSound(type): Handles 'open', 'close', 'minimize' variants.
- open: sweep 440 to 880 Hz over 0.1s, gain 0.12 decays over 0.15s
- close: sweep 880 to 220 Hz over 0.1s, gain 0.1 decays over 0.12s
- minimize: single 660 Hz, gain 0.08 decays over 0.06s
Wraps in try/catch.

WINDOW MANAGER SECTION:

Class WindowManager:

Constructor:
- this.windows = new Map()  // appId string -> { el, minimized, title, icon }
- this.zCounter = 300
- this.layer = document.getElementById('windowLayer')
- this.pillsContainer = document.getElementById('taskbarPills')

Method createWindow(appId, title, icon, contentEl, opts = {}):
1. If this.windows.has(appId), call this.restoreWindow(appId) and return.
2. Compute w = opts.width || 820, h = opts.height || 560.
3. Compute x = center with ±20px random jitter, y = center minus 14px with ±20px jitter.
4. Create div.win95-window.focused, set style left/top/width/height/z-index.
5. Set dataset.appId = appId.
6. Build window structure using DOM methods (NOT innerHTML for security):
   a. Create div.win95-titlebar with dataset.drag = 'true'
   b. Inside titlebar: span.win95-titlebar-icon (textContent = icon emoji),
      span.win95-titlebar-text (textContent = title),
      div.win95-titlebar-btns containing 3 button.win95-btn:
        button.min-btn (textContent '_'), button.max-btn (textContent '□'),
        button.close-btn (textContent '✕')
   c. Create div.win95-content with id = 'win-content-' + appId
   d. Create div.win95-statusbar with two spans (textContent title and 'Ready')
   e. Append titlebar, content, statusbar to win in order
7. If contentEl is provided, win.querySelector('.win95-content').appendChild(contentEl).
8. this.layer.appendChild(win).
9. this.windows.set(appId, { el: win, minimized: false, title, icon }).
10. Call this._addPill(appId, title, icon).
11. win.addEventListener('mousedown', () => this.focusWindow(appId)).
12. Call this._makeDraggable(win, win.querySelector('.win95-titlebar')).
13. Wire min-btn click: stopPropagation, playWindowSound('minimize'), this.minimizeWindow(appId).
14. Wire max-btn click: stopPropagation, this._toggleMaximize(appId).
15. Wire close-btn click: stopPropagation, playWindowSound('close'), this.closeWindow(appId).
16. Call playWindowSound('open').
17. Return win.

Method _addPill(appId, title, icon):
1. Create div.taskbar-pill.active with id = 'pill-' + appId.
2. Set textContent to icon + ' ' + title (max ~15 chars for title portion).
3. Set title attribute for tooltip.
4. Add click listener: if entry.minimized call restoreWindow, else call minimizeWindow.
5. Append to this.pillsContainer.

Method focusWindow(appId):
1. Remove 'focused' from all .win95-window elements.
2. Get entry from this.windows. If exists and not minimized: add 'focused', increment and set z-index.
3. Remove 'active' from all .taskbar-pill elements.
4. Add 'active' to pill element with id 'pill-' + appId.

Method minimizeWindow(appId):
1. Get entry. Set entry.minimized = true. Add 'minimized' to entry.el.
2. Get pill and remove 'active' class.

Method restoreWindow(appId):
1. Get entry. Set entry.minimized = false. Remove 'minimized' from entry.el.
2. Call focusWindow(appId).

Method closeWindow(appId):
1. Get entry. If not found, return.
2. Get contentArea (.win95-content). Find any .panel child inside it.
3. If panel found: remove 'open' class, clear panel.style.cssText, move it back to document.body
   using document.body.appendChild(panel).
4. Remove entry.el from DOM.
5. this.windows.delete(appId).
6. Remove pill element from DOM.

Method _toggleMaximize(appId):
1. Get entry and win element.
2. If win.dataset.maximized === 'true': restore win.style.cssText from win.dataset.prevStyle,
   set win.dataset.maximized = 'false'.
3. Else: save win.style.cssText to win.dataset.prevStyle,
   set win style to left:0, top:0, width:100%, height:calc(100% - 28px), z-index incremented,
   set win.dataset.maximized = 'true'.

Method _makeDraggable(win, handle):
1. On handle mousedown (skip if target has .win95-btn class):
   Record startX/Y, startL = parseInt(win.style.left)||0, startT = parseInt(win.style.top)||0.
2. On document mousemove: update win.style.left and win.style.top based on delta.
3. On document mouseup: remove mousemove and mouseup listeners.
4. Call e.preventDefault() on the mousedown event.

Instantiate: const wm = new WindowManager();
