IMPORTANT: All animations start from display:none implicit state — createWindow() already

Per-app animations (implement ALL 7):

'paper' — Pixel scatter dissolve:
- Get all .paper-section, .paper-abstract, .timeline-item elements inside el
- gsap.set(el, { opacity: 0 }) then gsap.to(el, { opacity: 1, duration: 0.05 })
- For each child element, gsap.fromTo with random x: (Math.random()-0.5)*200,
  y: (Math.random()-0.5)*200, opacity: 0, scale: 0.3
  To: x: 0, y: 0, opacity: 1, scale: 1, duration: 0.5 + Math.random()*0.3,
  ease: 'power3.out', stagger not used (each has own random delay: Math.random()*0.2)
- If no child elements found, fall back to elastic bounce (same as explorer)

'pres' — 3D flip on Y axis:
- Set parent perspective: el.style.perspective = '1200px'
- gsap.fromTo(el, { rotationY: -90, opacity: 0, transformOrigin: 'left center' },
  { rotationY: 0, opacity: 1, duration: 0.5, ease: 'power2.out' })

'exp' — Glitch effect (chromatic shift flicker):
- gsap.timeline() sequence:
  - set el opacity 1, x 0
  - to el: x -6, duration 0.04, repeat 4, yoyo true, ease 'none'
  - fromTo el: { filter: 'hue-rotate(0deg)' }, { filter: 'hue-rotate(360deg)', duration: 0.3, ease: 'none' }
  - then: { filter: 'none', x: 0 }
- Also animate the window el: gsap.fromTo(el, { scaleX: 1.04, scaleY: 0.96 },
  { scaleX: 1, scaleY: 1, duration: 0.3, ease: 'elastic.out(1, 0.5)' })

'terminal' — Matrix rain burst:
- First set: gsap.set(el, { opacity: 0, scale: 0.95 })
- Create 8 temporary span elements, position them absolutely within el,
  each with random column position and green text showing random characters (use 'ABCDEF0123456789')
  Each span: position absolute, top 0, left (col*12)+'px', color #33ff33,
  font-family Space Mono, font-size 12px, pointer-events none, z-index 1000
  Append each to el. Generate 8-12 random chars as textContent.
- GSAP sequence on each rain span: gsap.fromTo(span, {y: -20, opacity: 1},
  {y: el.clientHeight, opacity: 0, duration: 0.4 + Math.random()*0.3, ease: 'none',
   onComplete: () => span.remove() })
- After 0.3s: gsap.to(el, { opacity: 1, scale: 1, duration: 0.2, ease: 'power2.out' })

'explorer' — Elastic bounce from center:
- gsap.fromTo(el, { scale: 0.3, opacity: 0, transformOrigin: 'center center' },
  { scale: 1, opacity: 1, duration: 0.6, ease: 'elastic.out(1.2, 0.5)' })

'notepad' — Typewriter reveal:
- gsap.fromTo(el, { opacity: 0 }, { opacity: 1, duration: 0.1 })
- Find textarea inside el (querySelector('textarea'))
- If found: save original text, clear it, then type it back char by char using setInterval
  at 20ms per char. Total typing time should be ~1.5 seconds max for long text.
  (This means batch 2-3 chars per interval tick if text is long)
- If no textarea found: fall back to elastic bounce

'recycle' — Items fall in from top:

- If no children found: gsap.fromTo(el, { y: -20, opacity: 0 }, { y: 0, opacity: 1, duration: 0.3 })
